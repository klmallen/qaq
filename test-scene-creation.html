QAQ Game Engine - Scene Creation Test

这个测试页面用于验证场景创建功能是否正常工作。

测试说明

请按照以下步骤测试场景创建功能：

  1. 打开 QAQ Game Engine 编辑器 (http://localhost:3000/editor)
  2. 点击左侧场景树面板中的 "Create Scene" 按钮
  3. 检查是否创建了新场景并显示在场景树中
  4. 检查顶部是否出现了新的场景标签页
  5. 点击顶部标签栏的 "+" 按钮创建更多场景
  6. 验证场景之间的切换是否正常

预期结果

✅ 场景创建功能应该包含： - 点击 "Create Scene" 按钮后立即创建新的 3D 场景 - 场景自动命名为 Scene1, Scene2, Scene3 等 - 每个场景包含一个根节点 (Node3D) 和一个子节点 (MeshInstance3D) - 新场景标签页出现在顶部标签栏 - 场景树显示新创建的节点层次结构 - 可以在不同场景标签页之间切换 - 3D 视口显示默认的立方体网格
